Large Display Alarm Annunciator (대형 디스플레이 경보창)

ZERA-DSPANN2601

Overview

ZERA-DSPANN2601 제품은 관제실 또는 제어실의 대형 디스플레이용 경보 표시 시스템입니다.
일반적으로 대형 모니터를 제어실 벽면에 매립 또는 벽결이 타입으로 설치하며, 조작부인 Switch Panel은 제어 콘솔 데스크에 설치됩니다.


Advantage
  • 경보 추가/변경 시 경보창 공사 불필요
  • 유연한 경보 램프 및 명칭 변경, 크기 조정
  • 경보 식별 효과 확장성

System Configuration

제품 구성

ZERA-DSPANN2601은 경보 데이터를 수집하는 AlarmDataLink와 경보를 관리하고 대형 화면에 전송하는 AlarmDisplayMaster, 콘솔 데스크에 설치되는 조작 Switch Panel로 구성되어 있습니다.

• Software
    - AlarmDataLink: 제어 설비(DCS/PLC)의 경보 데이터 수집 및 AlarmDisplayMaster에 전송
    - AlarmDisplayMaster: AlarmDataLink로 부터 수신된 경보 신호를 각 대형 화면에 전송
• Hardware
    - Large Display: 대형 경보 표시 장치
    - Switch Panel: 조작 버튼(Silence, Ack, Reset, Test)
